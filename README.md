# Final Project 
### Chili Pests Detection System Using Yolo with tflite format
Chili Pests Detection System adalah aplikasi mobile berbasis Flutter yang dirancang untuk membantu petani mengidentifikasi hama pada tanaman cabai secara real-time. Proyek ini menggunakan model YOLOv8 yang dioptimalkan ke format TensorFlow Lite (TFLite) agar dapat berjalan dengan efisien dan ringan di perangkat smartphone.

## 🚀 Kapabilitas Sistem

| Fitur | Deskripsi |
| :--- | :--- |
| **Model YOLOv8** | Arsitektur state-of-the-art untuk deteksi hama yang akurat. |
| **Integrasi TFLite** | Konversi model untuk performa maksimal di perangkat mobile. |
| **Multi-Class Support** | Identifikasi spesifik untuk Kutu Daun, Kutu Kebul, dan Thrips. |
| **UI/UX Flutter** | Antarmuka aplikasi yang modern, bersih, dan mudah digunakan. |
| **Optimasi Gambar** | Pra-pemrosesan citra otomatis untuk meningkatkan akurasi deteksi. |

## 🛠️ Tech Stack
* **Model:** YOLOv8 (Ultralytics)
* **Format:** TFLite (FP16/INT8 Quantization)
* **Framework:** Flutter
* **Tools:** Roboflow (Dataset Management), Google Colab (Training)
